# Habitier

## 설명
- 습관 관리 및 공유를 하는 서비스입니다.

## 구현된 기능
- 회원가입 / 탈퇴
- 로그인 / 로그아웃
- 유저 정보 보기
- 이메일 코드 요청 / 인증
